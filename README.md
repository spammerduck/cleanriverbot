# CleanRiver Bot 🌊🤖

CleanRiver Bot is a futuristic autonomous robot boat designed to collect plastic waste from rivers, lakes, and oceans using smart robotics and renewable energy.

The project combines environmental protection, robotics, and innovation to create a cleaner and more sustainable future.

---

## ✨ Features

* Autonomous water navigation
* Plastic waste collection system
* Ultrasonic obstacle detection
* Solar-powered charging
* LED & OLED smart display
* Waterproof robotic design
* Bluetooth / mobile control support
* Futuristic eco-tech website UI

---

## 🌍 Purpose

CleanRiver Bot helps reduce water pollution by collecting floating plastic waste before it reaches oceans and harms marine life.

The project promotes:

* Sustainability
* Clean energy
* Smart robotics
* Environmental awareness

## 📸 Project Idea
